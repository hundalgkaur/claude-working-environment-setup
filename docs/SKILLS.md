# Recommended skills

Curated skills matched to this project. **Review/scan any third-party skill before installing**
(they are executable instructions). Offensive security skills (red-team / pentest) are for
authorized engagements only.

## Knowledge graph — graphify (default)

Run `/graphify` on this repo (code + docs) to get a clustered knowledge graph (HTML + JSON +
audit report). Useful for onboarding, architecture review, and spotting hidden coupling. Keep it
in your default toolkit for any non-trivial project.
